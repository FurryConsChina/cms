# FCC CMS

## Development

Make sure you already use right node version and yarn, if you not sure, please run:

```bash
nvm use
corepack enable
```

Install the dependencies:

```bash
yarn install
```

## Get Started

Start the dev server:

```bash
yarn dev
```

Build the app for production:

```bash
yarn build
```

Preview the production build locally:

```bash
yarn preview
```
